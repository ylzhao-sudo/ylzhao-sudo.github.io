6、SSRC

The SSRC is a synchronous sampling rate converter whose purpose is to convert a signal from one input sampling frequency to another specific frequency without changing the signal.

For example, at a sampling frequency of 44.1kHz, a 1kHz signal can be converted to a sampling frequency of 192kHz, assuming the signal is still 1kHz.

11、Does capture (2/6/8/12/16 channels) from board to Linux works without errors and with good audio quality?（choose ALSA output）

Note that the arecord parameter needs to match the number of channels output by the OM configuration and the stream sampling rate.

The default output of the Audio Frame is set to DAC and ALSA. Note that the Release SDCard image was developed for the reference design board, so the AK4458 DAC is automatically configured. From the point of view of Linux audio streaming, the audio framework is viewed as an ALSA device.ALSA’s Audio Framework provides a PCM interface that supports capture from 32 kHz to 192 kHz, up to 16 channels and 32 bit formats.

Capturing audio on Linux can be done with the arecord command：

Use the “record” command to record for a few seconds and then exit，ctrl c.

Then copy it to a USB flash drive, plug the USB flash drive into the pc and play to see if it is just recorded audio.

2、Alsa-cpp-output

Linux：

afrun.sh /dev/stdin

PUT pipeline0/pipeline.elt/0/state=NULL

PUT pipeline0/pipeline.elt/0/src_device=hdmi-input

PUT pipeline0/pipeline.elt/0/state=PLAYING

GET pipeline0/pipeline.elt/0/src_device

LK:

ppp cmd “PUT pipeline0/pipeline.elt/0/state=NULL”

ppp cmd “POST Element=pipeline0/alsa_extractor.elt/alsa_extractor0”
ppp cmd “POST Link=pipeline0/dts_virtualx.elt/dts_virtualx0&pipeline0/alsa_extractor.elt/alsa_extractor0”
ppp cmd “PUT pipeline0/alsa_extractor.elt/alsa_extractor0/enable=1”
ppp cmd “PUT pipeline0/alsa_extractor.elt/alsa_extractor0/out_chans=2”
ppp cmd “PUT pipeline0/alsa_extractor.elt/alsa_extractor0/out_format=32”
ppp cmd “PUT pipeline0/pipeline.elt/0/ppa_head=latency0&ppa_tail=alsa_extractor0&ppa_chunk_size=256&ppa_chunk_bitwidth=32&ppa_enabled=1”

ppp cmd “PUT pipeline0/pipeline.elt/0/om_config_path_id=4&om_device=alsa-output&om_chans=2”

ppp cmd “PUT pipeline0/pipeline.elt/0/state=PLAYING”
