ARC & eARC

Created2021-05-05|Updated2021-05-10
|Post Views:

（1）Brief description

Basic function test to make sure the EP (edge processing) basically works on the Audio board.
For example: eARC, ARC, audio transition, etc.

（2）Keywords

eARC, ARC & input source switch

（3）Operation steps

For Audio Board

  1. Make sure the devices are connected correctly

    • Boot the board, and check if there is any error/crash log during booting process on both sides.
    • jinit.sh
jstart.sh pp_sample

  2. Switch the TV output mode to the corresponding board output mode

  3. Start the test

    • Use Vlc player to play any audio, and check if different format audio can play normally and transition without error.

For Soundbar Devices

1. The first way of connection:

  • PC → Soundbar: HDMI-in
  • Soundbar → TV: eARC
  • PC (VLC) - Source output: Corresponding board
  • TV - Source output: Soundbar
  • Soundbar - Source: HDMI

2. The second way of connection:

  • PC → TV: HDMI
  • TV → Soundbar: eARC
  • TV - Source in: PC
  • Soundbar - Source: TV-ARC
